Far Resolver

Herramienta PowerShell de proposito general

Con esta herramienta se pretende crear una recopilacion de soluciones en PowerShell para Windows, para abarcar todo ripo de soluciones y herramientas para el Sistema Operativo de Microsoft.

Se pretende que sea una herramienta de proposito general, que pueda ser usada por cualquier persona, y que sea facil de usar.

Ademas de poder ofrecer y abarcar todo tipo de soluciones

Github del proyecto

Sientete libre de aportar tus ideas en el apartado Issues del proyecto

PowerDownloader

Herramienta PowerShell para descargar contenido multimedia de diferentes plataformas

Con esta herramienta, permite al usuario usar PowerShell para descargar contenido multimedia procesado de internet

Con la ayuda de un fork de Youtube-DL y FFMPEG, se puede descargar tanto videos, como musica de Youtube, entre otras plataformas de contenido en red

Github del proyecto

PowerShell APIS

Coleccion de diferentes REST APIs implementadas en PowerShell

Este repositorio es una coleccion de REST APis llevadas a cabo en PowerShell

Se pretende enseñar y ayudar a otros usuarios sobre como implementar en un proyecto PowerShell, una REST API

Poco a poco se ira expandiendo el contenido
